<h2>Overview</h2>
<p>Etrix is a powerful Angular-based web application designed to streamline development and enhance user experience through a modern architecture.</p>
<p><strong>Why Etrix?</strong></p>
<p>This project empowers developers to build scalable and maintainable applications with its robust features. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🎨 <strong>TypeScript Configuration:</strong> Ensures strict type checking and consistent file handling, enhancing code quality.</li>
<li class="my-0">🌈 <strong>Tailwind CSS Integration:</strong> Provides a customizable design system for responsive and adaptive layouts.</li>
<li class="my-0">🚀 <strong>Angular Architecture:</strong> Streamlined project structure with built-in support for internationalization and testing.</li>
<li class="my-0">🔄 <strong>Dynamic Components:</strong> Engaging UI elements like carousels and responsive grids enhance user interaction.</li>
<li class="my-0">🧪 <strong>Unit Testing Framework:</strong> Ensures reliability and maintainability of components through comprehensive testing.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> TypeScript</li>
<li class="my-0"><strong>Package Manager:</strong> Npm</li>
</ul>
<h3>Installation</h3>
<p>Build Etrix from the source and intsall dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/italomangueira/Etrix
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd Etrix
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">❯ npm install
</code></pre>
<h3>Usage</h3>
<p>Run the project with:</p>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">npm start
</code></pre>
